# Prompt 04 — Catalog Schema and Shared JS

Design and implement the `catalog.json` schema and the shared JavaScript module `app.js`.

## catalog.json schema

The file is generated by the GitHub Action and read by the browser. It needs:

```json
{
  "baseUrl": "https://pub-xxxx.r2.dev",
  "hero": [
    { "filename": "hero1.jpg", "width": 4000, "height": 2667 }
  ],
  "categories": [
    {
      "id": "aviation",
      "name": "Aviation",
      "cover": "aviation/cover.jpg",
      "flat": false,
      "albums": [
        {
          "id": "oshkosh-2024",
          "title": "EAA AirVenture Oshkosh 2024",
          "date": "2024-07",
          "location": "Oshkosh, Wisconsin",
          "folder": "aviation/oshkosh-2024",
          "cover": "aviation/oshkosh-2024/cover.jpg",
          "photos": [
            {
              "filename": "img001.jpg",
              "width": 3000,
              "height": 2000,
              "caption": "",
              "exif": {
                "camera": "NIKON Z9",
                "lens": "NIKKOR Z 100-400mm f/4.5-5.6 VR S",
                "focal_length": "400mm",
                "aperture": "f/5.6",
                "shutter_speed": "1/2000s",
                "iso": "ISO 800"
              }
            }
          ]
        }
      ]
    }
  ]
}
```

## app.js

Create `app.js` as an ES module with these exports:

- `fetchCatalog()` — fetch and cache `/catalog.json`
- `fullUrl(base, path)` — constructs `${base}/${path}`
- `thumbUrl(base, path)` — constructs `${base}/_thumbs/${path}`
- `fadeInOnLoad(img)` — adds `.loaded` class when the image loads (or immediately if already complete)
- `formatDate(str)` — converts `"2024-07"` → `"July 2024"`, `"2024"` → `"2024"`
- `hydrateCategoryTiles(catalog)` — finds all `.category-tile` elements, reads the `?cat=` param from their existing `href`, injects a cover `<img>` element
- `makeAlbumTile(album, catId, catName, baseUrl)` — creates an `.album-tile` anchor element with cover image, title, and subtitle (category name · formatted date)
- `getLatestAlbums(catalog, n)` — returns the `n` most recent albums across all categories, sorted by date descending
- `renderLatestAlbums(catalog)` — populates `#latest-albums` with the four newest albums
- `renderAlbumGrid(catalog, catId, container)` — populates a container with all album tiles for a given category
