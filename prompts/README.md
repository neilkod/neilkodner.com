# Prompts

This directory is a build log — each file is the prompt used to construct a feature of the site in a Claude Code session, in order.

They document *what was asked for* at each step, not the current state of the code. Some details (category names, folder structures) reflect the project as it existed when the prompt was written and may differ from today.

## Purpose

- **Historical record:** why decisions were made and in what order
- **Generalization recipe:** a reusable sequence for building a similar photography portfolio from scratch

## Adding new entries

When a significant feature is added, create the next numbered file (`20_feature_name.md`) describing what was asked for. Keep the original prompt text; don't retroactively edit earlier files.
