# Prompt 17 — About Page

Build `about/index.html` — a simple personal bio page.

## Content

- Site navigation (same as all other pages, `aria-current` on About)
- `<h1>About</h1>`
- A few short paragraphs:
  - I'm a photographer based in Los Gatos, California
  - By day I'm an engineering manager at Airbnb
  - The photography categories I shoot: aviation, hockey, birds, and places
  - Brief mention of gear or approach (keep it short — this is a portfolio, not a blog)
- Contact/social links:
  - Email: `nkodner@gmail.com`
  - LinkedIn: `https://www.linkedin.com/in/neilkodner`
- Footer (same as all pages)

## Meta tags

- `<title>About — Neil Kodner</title>`
- `<meta name="description">` — one sentence about the photographer
- Open Graph tags mirroring the description

## Style notes

- No special CSS needed beyond what's in `style.css` already
- The about page should feel consistent with the rest of the site — same dark background, same nav, same footer
- Keep the copy minimal. On a photography portfolio, the photos do the talking.
